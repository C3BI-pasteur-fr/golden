
=head1 NAME

=over 4

=item B<goldin> - Databank flat files indexer

=back

=head1 SYNOPSIS

=over 4

=item B<goldin> [B<-ahiq>] [B<-d> F<dir>] F<dbase> F<file> ...

=back

=head1 OPTIONS

Following command line options are allowed:

=over 4

=item -a

Generate accession number indexes.

=item -d F<dir>

Use alternate directory F<dir> (instead of default F<dbase>), to
retrieve the databanks entries. This option might be useful to index
databanks subsets.

=item -h

Usage display.

=item -i

Generate entry names indexes.

=item -q

Be quiet. Do not display some warning messages.

=back

=head1 DESCRIPTION

B<goldin> program update/create databanks flat files indexes, which
can be used by B<golden> program to make fast databanks entries
queries.

By default, both entry names and accession numbers indexes are
generated (same as setting B<-a> and B<-i> flags) in the current
working directory.

Indexed databank consist in 4 objects: 3 files generated by B<goldin>
program, and 1 directory holding flat files.

=over 4

=item B<foo.idx>

File containing entry names indexes.

=item B<foo.dbx>

File containing databank sections list.

=item B<foo.acx>

File containing accession numbers indexes.

=item B<foo/>

Directory holding databank flat files, if not set otherwise with B<-d>
flag.

=back

=head1 ENVIRONMENT

=over 4

=item TMPDIR

If set, this directory is used to store temporary generated files;
instead of default F</tmp>.

=back
